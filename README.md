Mock API data is hosted in Postman.

You can run the app in one of two ways:

- Use the "Live Server" VS Code extension.

- Serve the project with http-server:
  - (optional) Install: `npm install -g http-server`
  - Start: `npx http-server -p 3000`
  - Or, if installed globally: `http-server -p 3000`
